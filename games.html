Games

Below, you will find download links to some of the games we have made, as well as the platforms or software they run on. Enjoy!
for DOS...
For DOS games, you should move ALL files to a storage medium (such as a floppy disk), reboot your computer, and then directly boot into the storage medium. This is the easiest way to ensure that a game will run as absolutely intended.
for NXT...
For games meant to be played on an NXT, Download Enchanting to your computer, open it, then in the editor, open the project file, and then send it to your NXT, then, boot into LEJOS, go to files, and then open the first file.
for BROWSER...
Simply open the link. these can be played on any modern browser that supports Javascript! :D
